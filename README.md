# MLOps project based on MNIST classifier
## Installation

```
# Clone github repo
# For the latest code:
git clone https://github.com/Mavic-ux/mlops_mnist.git

# Change directory into project directory
cd mlops_mnist

# Activate virtual environment
poetry shell

# Install mlops_mnist from source
poetry install

# Run training
python3 train.py

# Run inference
python3 infer.py

# Model and predictions are stored in checkpoints/ directory be default

# Exit the virtual environment
exit
```
